Περιγραφή:
Το πρόγραμμα προσομοιώνει την εξυπηρέτηση μιας ημέρας ενημέρωσης γονέων.
Αρχικά ένα τυχαίο πλήθος από τους συνολικούς γονείς μπαίνουν, ένας-ένας, στην ουρά.
Για κάθε καθηγητή επιλέγεται ο γονέας που προηγείται, αν υπάρχει, και αφαιρείται από την ουρά.
Για κάθε γονέα που αφαιρείται, ένα τυχαίο πλήθος των συνολικών γονέων προστίθεται στην ουρά.
Κάθε γονέας που επιλέγεται είναι πλέον υπό ενημέρωση από τον εκάστοτε καθηγητή.
Η ενημέρωση ολοκληρώνεται και όσοι γονείς δεν επιθυμούν να ενημερωθούν από άλλον καθηγητή αποχωρούν (εκτυπώνεται: Parent with id <ταυτότητα_γονέα> is leaving).
Οι υπόλοιποι επιστρέφουν στην ουρά.
Η επιστροφή ενός γονέα στην ουρά μπορεί να γίνει είτε σωστά, οπότε οι υπόλοιποι γονείς τον επιδοκιμάζουν (εκτυπώνεται: Well done! <ταυτότητα_γονέα>),
είτα παράτυπα οπότε οι υπόλοιποι γονείς διαμαρτύρονται (εκτυπώνεται: You should go to the end of the line! <ταυτότητα_γονέα>).
Η διαδικασία αυτή επαναλαμβάνεται μέχρι όλοι οι γονείς να αποχωρήσουν από την ουρά, όντας πλήρως ενημερωμένοι.

Μεταγλώττιση:
Παράδειγμα μεταγλώττισης:
g++ -o Parents_Briefing Parents_Briefing.cpp class_Parent.cpp class_Parent_List.cpp class_Teacher.cpp

Εκτέλεση:
Παράδειγμα εκτέλεσης: ./Parents_Briefing
Insert the number of teachers: 3
Insert the number of parents: 5
...

Κατά την εκτέλεση του προγράμματος θα πρέπει, ακολουθώντας τα μηνύματα προτροπής, να δωθούν από το standard input τα παρακάτω με την εξής σειρά:
    N: Το συνολικό πλήθος καθηγητών που θα λάβουν μέρος στην ενημέρωση
    K: Το συνολικό πλήθος γονέων που θα λάβουν μέρος στην ενημέρωση
